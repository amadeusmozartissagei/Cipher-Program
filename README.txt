# Cara Menjalankan Program

1. Pastikan Anda memiliki Python dan Flask terinstal.
2. Jalankan perintah berikut untuk menginstal Flask:
   pip install Flask
3. Jalankan aplikasi dengan perintah:
   python app.py
4. Buka browser dan akses http://127.0.0.1:5000/
5. Masukkan teks, kunci, dan pilih jenis cipher untuk mengenkripsi.
6. Untuk Hill Cipher, masukkan kunci dalam format berikut:
   - Untuk matriks 2x2: "6 24 1 13"
   - Untuk matriks 3x3: "1 2 3 0 1 4 2 0 1"
